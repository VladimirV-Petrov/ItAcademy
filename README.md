Module 9
---
Создать 2 класса: Forest, Tree  
Класс Forest содержит в себе набор объектов типа Tree  

Класс Tree:  
- в конструкторе класса передается имя дерева и тип дерева (тип дерева - это enum из нескольких пород деревьев)  
- реализован конструктор копирования  
- в классе имеется счетчик, который хранит сколько всего деревьев было создано  
- каждому созданному дереву в конструкторе устанавливается уникальный id - как вариант текущее количество деревьев + 1  
- в классе реализован метод wind() - который выводит на экран имя дерева и его id  
- константные геттеры getID и getName  

Класс Forest:  
- хранит в себе список (массив, вектор, лист...) объектов типа дерева  
- реализованы функции  
    - getTreesNumber() - получение количества деревьев в лесу  
    - growUp(...) - (вырастить дерево) добавление дерева в список  
    - cutAll() - (вырубить все деревья) Очистка списка деревьев  
    - operator+ - (два леса срослись в один большой) - нужно соединить все деревья обоих лесов и вернуть новый лес  

main:  
должно быть продемонстрировано создание деревьев и добавление их в леса, а также слияние лесов и их вырубку

Module 13

Шаблонный класс ServiceStation:  
Отвечает за хранение списка машин, их добавление и ремонт.  
Внутри класса ServiceStation инкапсулирован шаблонный класс Garage, который хранит отремонтированные машины.  
Методы:  
-AddCar: добавляет машину в СТО, если количество машин меньше лимита. Если лимит превышен, выводит сообщение об ошибке.  
-RepairAll: чинит все машины в СТО (вызывает метод Repair у каждой), перемещает их в гараж и очищает СТО.  
-DisplayGarage: выводит список всех машин, отремонтированных и помещённых в гараж.  
Шаблонный класс Garage:  
Отвечает за хранение отремонтированных машин.  
Методы:  
-AddToGarage: добавляет машину в гараж.  
-DisplayAll: выводит информацию обо всех машинах в гараже.  
Классы автомобилей (PassengerCar и CargoCar):  
Поля:  
-std::string name — название машины.  
-bool isRepaired — состояние машины (починена или нет).  
Методы:  
-Repair: устанавливает isRepaired = true.  
-Перегруженный оператор << для вывода информации о машине.  

Программа:  
Создать два объекта класса ServiceStation:  
Один для легковых автомобилей (PassengerCar).  
Второй для грузовых автомобилей (CargoCar).  
Добавить по 6 машин в каждую СТО (при лимите 5 машин).  
Попытаться починить машины в каждой СТО:  
Отремонтированные машины переместить в гараж.  
Вывести информацию о машинах до и после ремонта, а также о содержимом гаража.
